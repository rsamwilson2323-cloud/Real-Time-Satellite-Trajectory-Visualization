# 🛰️ Real-Time Satellite Trajectory Visualization

An interactive **real-time satellite trajectory visualization** built with **React, TypeScript, Vite, Tailwind CSS, and modern web technologies**. The application provides an engaging 3D interface for exploring satellite motion, orbital paths, and space visualization with smooth rendering and responsive controls.

---

## ✨ Features

* 🛰️ Real-time satellite trajectory visualization
* 🌍 Interactive Earth and orbital path rendering
* 📡 Dynamic satellite position updates
* 🎮 Smooth zoom, pan, and rotation controls
* ⚡ High-performance rendering with Vite
* 🎨 Modern responsive UI using Tailwind CSS
* 📱 Cross-platform browser support
* 🌌 Space-themed visualization experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/rsamwilson2323-cloud/Real-Time-Satellite-Trajectory-Visualization.git

cd Real-Time-Satellite-Trajectory-Visualization
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
